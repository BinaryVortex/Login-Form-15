# Login Form (Login-Form-15)

A clean, responsive login form built with HTML, CSS and JavaScript.

This repository contains a single-page login form demo with a polished UI and simple client-side validation. It's ideal as a starter component for personal projects, portfolios, or UI practice.

## Preview

![Login Form Screenshot](./Screenshot%202024-05-28%20020949.png)

## Features

- Modern, minimal design
- Responsive layout for desktop and mobile
- Simple client-side validation (JavaScript)
- Easy to customize colors, spacing and form fields

## Files

- `index.html` — main page containing the form markup, styles and scripts.
- `Screenshot 2024-05-28 020949.png` — demo screenshot used in this README.

## How to use

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-15.git

2. Open `index.html` in your browser (double-click or use Live Server).

3. Customize the HTML/CSS as needed.

## Customize

- To change colors or spacing, edit the CSS inside `index.html` or move styles to a separate `.css` file.
- To add real authentication, replace the demo validation with your backend API calls.
